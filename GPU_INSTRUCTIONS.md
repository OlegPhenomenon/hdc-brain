# Инструкции для продолжения обучения на GPU-сервере (vast.ai)

## Контекст проекта

Это реализация **теории сознательных агентов Дональда Хофмана** как языковой модели.
Рой субагентов с марковским ядром взаимодействия коллективно генерирует текст Шекспира.

### Текущее состояние (v6 — Multi-Channel Swarm)
- **Лучший BPB: 2.1587** (веса потеряны при смене конфигурации)
- **Архив v4 модели: archive/best_swarm.pt** (BPB ~2.18, архитектура v4 — одноканальная)
- **Цель: BPB ≤ 2.0**
- Датасет: Tiny Shakespeare (1.1M символов, vocab 65)
- Параметров: ~1.14M

### Архитектура v6
- 32 агента, STATE_DIM=256, 2 канала коммуникации (быстрый k=10 + медленный k=4)
- GRU-гейтинг для обновления состояний агентов
- Сенсорная подгруппа (8 агентов получают вход)
- Взвешенный консенсус (обучаемые веса важности)
- Small-world графы связей (Watts-Strogatz)

## Что менять в train.py для GPU-сервера

### Гиперпараметры — УВЕЛИЧИТЬ (GPU мощнее)
```python
# === ГИПЕРПАРАМЕТРЫ ДЛЯ GPU (RTX 4090) ===
N_AGENTS = 64           # Было 32 → больше агентов в рое
STATE_DIM = 256         # Оставить — хороший размер
N_SENSORY = 16          # Было 8 → больше агентов видят вход
N_CHANNELS = 2          # Оставить
N_INTERACTION_STEPS = 2 # Было 1 → глубже обсуждение агентов
MICRO_BATCH = 64        # Было 32 → GPU потянет
GRAD_ACCUM_STEPS = 2    # Эффективный батч = 128
SEQ_LEN = 256           # Было 128 → больше контекста
LEARNING_RATE = 5e-4    # Начать с нуля — стандартный LR
WARMUP_STEPS = 100
DROPOUT = 0.1
```

### Термо-паузы — УБРАТЬ (GPU не перегревается)
```python
# Закомментировать или удалить:
# THERMAL_PAUSE_EVERY = ...
# THERMAL_PAUSE_SEC = ...
# Все блоки с torch.mps.synchronize() и time.sleep()
# Все блоки с torch.mps.empty_cache()
```

### Таймер — УВЕЛИЧИТЬ
```python
signal.alarm(3600)  # 1 час или больше — GPU позволяет
```

### Device — автоматически подхватит CUDA
```python
# Строка уже есть в коде:
DEVICE = 'mps' if torch.backends.mps.is_available() else 'cpu'
if not torch.backends.mps.is_available() and torch.cuda.is_available(): DEVICE = 'cuda'
# На сервере будет DEVICE = 'cuda'
```

## Стратегия обучения на GPU

### Фаза 1: Обучение с нуля (1-2 часа)
- Параметры как выше (64 агента, SEQ_LEN=256)
- LR=5e-4 с cosine annealing
- Ожидаемый результат: BPB ~2.0-2.2

### Фаза 2: Точная подстройка (30-60 мин)
- Снизить LR до 1e-4
- Увеличить DROPOUT до 0.15 если train/val разрыв > 0.3
- Ожидаемый результат: BPB < 2.0

### Фаза 3: Эксперименты (если время есть)
- SEQ_LEN=512 — ещё больше контекста
- N_AGENTS=96 — ещё больше агентов
- N_INTERACTION_STEPS=3 — глубже обсуждение
- Добавить расширенный датасет (Шекспир + сонеты + библия)

## Важные правила

1. **НЕ удалять веса** — при смене архитектуры переносить в archive/
2. **results.tsv** — лог всех экспериментов, не затирать
3. **swarm_consciousness.log** — лог генераций роя, append-only
4. **chat.py** — при смене архитектуры обновить параметры в chat.py тоже
5. **best_swarm.pt** — сохраняется только при побитии рекорда BPB
6. **last_swarm.pt** — сохраняется всегда в конце сессии (для продолжения)

## Оценка прогресса

| BPB | Уровень |
|-----|---------|
| >3.0 | Шум, случайные символы |
| 2.5-3.0 | Появляются слова |
| 2.0-2.5 | Связные фразы, имена персонажей |
| 1.5-2.0 | Грамматически правильные предложения |
| <1.5 | Почти осмысленный Шекспир |
