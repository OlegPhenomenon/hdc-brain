# Проект II. Твой первый HDC-классификатор

---

## Что строим

Классификатор текстов на основе HDC-операций. Дали текст — определи категорию (спорт, политика, технологии). Без нейросетей, без градиентов — только Bind, Bundle и косинусная похожесть.

---

## Что тебе нужно знать

- **Bind (⊙)** — глава 6, секция "Операция 1: BIND — это принадлежит тому". Используем чтобы связать слово с его позицией.
- **Bundle (+)** — глава 6, секция "Операция 2: BUNDLE — собрать в кучу". Используем чтобы собрать все слова документа в один вектор.
- **Permute (ρ)** — глава 6, секция "Операция 3: PERMUTE — порядок имеет значение". Кодируем порядок слов.
- **Bipolar вектора** — глава 7, секция "32 бита — не всегда нужны". Работаем с ±1 вместо float.
- **STE** — глава 8, секция целиком. Пригодится если захочешь сделать обучаемую версию.
- **Косинусная похожесть** — глава 2, секция "Как измерить похожесть". Это наша метрика для классификации.

## Тесты

```python
# Обучающие данные
docs = {
    "спорт": [
        "гол мяч футбол матч команда победа".split(),
        "бег марафон финиш рекорд спортсмен".split(),
        "хоккей шайба лёд вратарь гол".split(),
    ],
    "технологии": [
        "код программа баг сервер деплой".split(),
        "процессор память диск сервер нагрузка".split(),
        "алгоритм данные модель обучение нейросеть".split(),
    ],
}

class_vectors, codebook = train_classes(docs)

# Классификация
assert classify("футбол матч гол".split(), class_vectors, codebook) == "спорт"
assert classify("сервер код деплой".split(), class_vectors, codebook) == "технологии"

# Вектор документа — bipolar
vec = encode_document("тест слово".split(), codebook)
assert set(np.unique(vec)).issubset({-1, 1})
```
