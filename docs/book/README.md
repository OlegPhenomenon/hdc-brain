# Думающие векторы: строим языковую модель с нуля

> Книга о том как работает HDC-Brain — языковая модель на 299 миллионов параметров,
> построенная не на трансформерах, а на принципах Hyperdimensional Computing.
>
> **Цель**: после прочтения ты сможешь реализовать такую архитектуру с нуля по памяти.
>
> **Стиль**: история → термин → код → формула → "вот что это буквально значит"

---

> Математика и код — инструменты, не цель.

## Как устроена каждая глава

```
1. История с конкретным персонажем
   ↓
2. Концепция — только после того как ты "почувствовал" проблему
   ↓
3. Код (минимальный рабочий пример)
   ↓
4. Формула — только после кода, с расшифровкой каждого символа
   ↓
5. Связь с HDC-Brain конкретно
   ↓
6. Задания — вопросы БЕЗ ответов, нужно исследовать самому
```

## Как устроен каждый проект

```
1. Что строим (конкретный результат)
   ↓
2. Почему это важно (связь с архитектурой)
   ↓
3. Спецификация: входы, выходы, ограничения
   ↓
4. Скелет кода (только сигнатуры функций, без реализации)
   ↓
5. Тесты для проверки (запусти — все должны пройти)
   ↓
6. Улучшения по желанию (исследовательские задачи)
```
