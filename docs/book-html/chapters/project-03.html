Проект: Мини-кодбук для русских слов

Обучаемый кодбук (codebook) — таблицу, которая превращает токены в HDC-вектора. Это центральный компонент HDC-Brain: без кодбука модель не может "видеть" слова.

Что строим

Обучаемый кодбук (codebook) — таблицу, которая превращает токены в HDC-вектора. Это центральный компонент HDC-Brain: без кодбука модель не может "видеть" слова.

В проекте II мы использовали случайные вектора для слов. Здесь — учимся делать вектора осмысленными: похожие слова должны получать похожие вектора.

Что тебе нужно знать

  • BPE-токенизация — глава 9, секция "Как работает BPE: пошагово". Нужно превратить текст в токены прежде чем строить кодбук.
  • Позиционное кодирование — глава 10, секция целиком. Кодбук хранит "смысл" токена, а позиция добавляется отдельно через Permute.
  • Codebook в HDC-Brain — глава 11, секция "Codebook — словарь как отпечатки пальцев". Это архитектурная основа того что мы строим.
  • Bind и Bundle — глава 6. Используем для обучения кодбука через контекст.
  • STE (Straight-Through Estimator) — глава 8, секция целиком. Нужен чтобы обучать bipolar вектора градиентами.
